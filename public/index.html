Voice Chunker

Tag dialogue by speaker, run find/replace, and split it into size-capped chunks.

▸ Splitting & voice options
A single space by default.
A switch = the speaker changing within a chunk. Blank = no limit.
{name} is replaced with the speaker. Dialogue follows after a space.

Script vs Prose: Script reads one line per speaker (NAME: dialogue); untagged lines become a narrator voice. Prose ignores tags and splits a wall of text on sentence boundaries — handy when there are no speakers.

Voices & switches (script only): a switch is when the speaker changes between two lines inside one chunk. Set a cap in the options, or leave it blank for no limit. The narrator doesn't count unless you enable it.

Continued lines: anything too long for a single chunk is cut on word boundaries and marked with …; in script mode each continuation re-opens with the speaker name so the right voice carries over.
